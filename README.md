## Comfy sloth interiors

## Build Setup

### .env vars

- REACT_APP_AUTH_DOMAIN=
- REACT_APP_CLIENT_ID=
- REACT_APP_STRIPE_PUBLIC_KEY=
- REACT_APP_STRIPE_SECRET_KEY=

### install dependencies

$ npm install

#### serve with hot reload at localhost:3000

$ npm run dev

### build for production and launch server

$ npm run build
$ npm run start
